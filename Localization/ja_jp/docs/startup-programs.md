# Microsoft for Startups とは

Microsoft for Startups は、スタートアップ向けに Azure クレジット、技術サポート、共同販売機会などを提供する支援プログラムです。

- 申請手順、必要条件、審査の流れ
- 申請時のポイントや注意事項

## 3. 特典の活用方法

- 技術サポートやリソースへのアクセス方法

- イベント、ミートアップ、学習リソース
