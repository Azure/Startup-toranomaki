# 如何联系微软支持？

如需联系 Azure 微软支持：

1. 访问 [Azure 门户](https://portal.azure.com/)
2. 点击左侧菜单的“帮助 + 支持”
3. 点击“创建支持请求”
4. 填写相关信息并提交

**提示：**
- 选择正确的严重级别并提供详细信息。
- 可在门户中跟踪请求进度。
- 社区支持可参考 [Microsoft Q&A](https://learn.microsoft.com/answers/topics/azure.html) 或 [Stack Overflow](https://stackoverflow.com/questions/tagged/azure)。
