# 安全与认证指南（中文）

本文件总结了保护 Azure 资源和应用的安全最佳实践。

---

## 1. 身份与访问管理
- Azure AD/Entra ID 基础
- RBAC（基于角色的访问控制）设置

## 2. 密钥与机密管理
- 使用 Azure Key Vault 管理密钥、机密和证书

## 3. API 安全
- CORS、限流、API 认证

## 4. 合规与数据保护
- 静态/传输加密
- 合规（如 GDPR）
